
- clone https://github.com/pencilresearch/midi to obtain the .csv files
    $ git clone https://github.com/pencilresearch/midi

- adjust the GIT_MIDI_PATH in convert_git.sh as required (must point to the cloned repo)

- run
    $ ./convert_git.sh
  to convert the .csv files to .msp format

- select / copy / move .msp files to parent directory (midi_synth_profiles/) as required
