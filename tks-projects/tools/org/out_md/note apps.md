note apps
=========


## whink
   - _can_** export text in PDFs**
   - no OCR
   - audio clips
   - annotate PDFs
   - slow undo


## good notes
   - can't export text !!
   - OCR is cumbersome
     must copy to clipboard first
   - image export


## notability
   - good OCR
   - intelligent shapes
   - can't export text !!
   - audio clips
   - embed and annotate PDFs
   - embed web pages
   - export PDF


## notes
   - no positionable text areas
   - basic shapes
     ruler, no autosnap
   - no OCR
   - no PDF


## nebo
   - excellent OCR
     as-you-write
   - export HTML with inlined images and SVGs
   - export PDF
   - no links !!
   - diagrams
   - images
      - doodles
   - LaTEX math
   - multi-column


## bear
   - attach any file
   - export zip
   - markdown
   - tags
   - images
   - annotate PDFs
   - no folders no shared files no OCR
